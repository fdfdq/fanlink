# Fanlink - 现代化浏览器起始页

Fanlink 是一个美观、高效、高度可定制的浏览器起始页，旨在为用户提供极致的上网体验和便捷的导航服务。

## ✨ 主要特性

### 🎨 现代化界面设计
- **网格布局视图**：图标在上、文字在下的清晰展示方式
- **响应式设计**：完美适配各种屏幕尺寸和设备
- **视觉优化**：清爽现代的界面，提供直观的视觉体验

### ⚡ 高效搜索体验
- **一键直达搜索**：输入关键词直接跳转搜索引擎
- **流程简化**：移除冗余中间页面，提升搜索效率
- **流畅自然**：优化后的搜索流程更加快速便捷

### 🔧 高度个性化
- **个性化设置面板**：集成边栏显示控制等核心功能
- **便捷操作**：告别繁琐设置，一键隐藏/显示分类栏
- **统一布局**：设置选项分类合理，操作直观便捷

### 📱 多平台支持
- **网页版**：在线访问，数据云端同步
- **桌面版**：本地应用，更稳定性能表现

## 🚀 快速开始

### 在线使用
直接访问官方网站：https://fdfdq.github.io/fanlink/fanlink.html

## 📥 安装与使用

1. **设置为浏览器主页**
   - 打开浏览器设置
   - 将主页设置为：`https://fdfdq.github.io/fanlink/fanlink.html`

2. **个性化配置**
   - 点击设置图标进入个性化面板
   - 调整布局、主题、边栏显示等选项
   - 添加常用网站收藏

3. **开始使用**
   - 使用搜索框快速搜索
   - 通过网格布局快速访问收藏网站
   - 使用便签功能记录重要信息
